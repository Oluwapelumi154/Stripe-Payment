# Types of Api Keys

# Publishable key

# Secret Key

# Restricted Key

# Webhook signing secret

# Publishable Key are used for making api calls on the client side to the stripe api

# Secret Key are used for making api calls from your server to the stripe api

# Secret key should be stored in safe file

# key Prefixes

# Publishable key starts with pk_id

# Secret key starts with sk_id

# Restricted key starts with rk_id

# WebHook signing secrets starts with whsec_id
